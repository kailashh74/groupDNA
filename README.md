# GroupDNA

GroupDNA is a Python-based WhatsApp chat analyzer that extracts meaningful insights from exported group chat files.

## Features
- Chat parsing and participant statistics
- Busiest day and busiest hour analysis
- Activity heatmap
- Most frequently used words
- Response time analysis
- Silent streak detection
- Personality archetypes for group members

## Tech Stack
- Python
- NumPy

## Usage

```bash
python "minor1(groupDNA).py" hostel_bois.txt
```

Replace `hostel_bois.txt` with your exported WhatsApp chat file.
